# The Preditorium Prophecy Website

Static website for *The Preditorium Prophecy: The Golden Statue* by Praanav J Thaakur.

Live site: https://preditorium.com

## Project Shape

This is a plain static site:

- `index.html` is the main page.
- `images/` contains the book cover, author image, favicon and app icons.
- `sample-chapter.pdf` is the free chapter sample.
- `robots.txt`, `sitemap.xml`, `llms.txt` and `site.webmanifest` support SEO, AEO and install metadata.

## Deploy

The repository is connected to Vercel project `sixideas/preditorium-prophecy`.
Pushes to `main` should deploy through Vercel.
